# Proceso para conseguir la user.txt:
Vamos a realizar un escaneo de puertos desde la maquina atacante:

  ```bash
sudo nmap -p- -open -sV -sC -sS --min-rate 5000 -n -Pn -vvv <ip-victima>
```
